Kelompok IaaS

Matkul : Cloud Computing

Cloud computing atau komputasi awan adalah model layanan yang memungkinkan pengguna untuk mengakses berbagai sumber daya teknologi seperti penyimpanan data, server, jaringan, perangkat lunak, dan aplikasi melalui internet. Dengan cloud computing, pengguna tidak perlu membeli dan mengelola infrastruktur IT fisik sendiri, melainkan menggunakan layanan yang disediakan oleh penyedia cloud secara on-demand (sesuai kebutuhan).

Praktek Bagian : IaaS (Infrastructure as a Service)

IaaS (Infrastructure as a Service) adalah salah satu model layanan dalam komputasi awan (cloud computing) yang menyediakan infrastruktur IT virtual melalui internet. Layanan ini memungkinkan pengguna untuk menyewa sumber daya komputasi seperti server, penyimpanan data (storage), jaringan, dan perangkat keras lainnya tanpa perlu membeli atau mengelola infrastruktur fisik sendiri. Contoh penyedia layanan IaaS adalah AWS EC2
