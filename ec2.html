AWS EC2 (Elastic Compute Cloud)

AWS EC2 adalah layanan komputasi berbasis cloud dari AWS yang menyediakan kapasitas server virtual (virtual machines) secara fleksibel, skalabel, dan on-demand. EC2 memungkinkan pengguna untuk menjalankan aplikasi di server virtual tanpa perlu membeli atau mengelola perangkat keras fisik.
